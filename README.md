# SimpleVoIP

Приложение для групповых видео- и аудиозвонков в реальном времени. Работает по схеме **WebRTC Full Mesh** (прямой обмен медиапотоками между браузерами) с использованием сигнального сервера на **C++ (Crow)**.

## Возможности
- **Изолированные комнаты**: Подключение по уникальным ссылкам.
- **Модерация**: Первый участник получает статус хоста с возможностью исключать других пользователей. При выходе хоста права передаются следующему.
- **Визуальные индикаторы**: Подсветка говорящего участника (анализ звука через Web Audio API) и отображение статусов микрофона/звука (Mute/Deafen) остальных пользователей.
- **Адаптивный интерфейс**: Динамическая подстройка сетки видео под количество участников (до 5 человек).
- **Смена камер**: Поддержка переключения камер на мобильных устройствах.

## Запуск

### Через Docker
```bash
docker-compose up --build
```
Интерфейс доступен на `http://localhost:8080`.

### Локальная сборка
Требуются CMake и компилятор с поддержкой C++17.
```bash
cmake -B build -S . -DCMAKE_BUILD_TYPE=Release
cmake --build build
```
Запуск исполняемого файла из директории `build` (например, `.\build\Release\webrtc_server.exe` для Windows).
